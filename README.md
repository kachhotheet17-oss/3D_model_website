# 3D House Model Designer

A responsive front-end website for designing and downloading basic 3D house models.

## Features

- Home page with live 3D house demo
- Login and signup modal using browser local storage
- Ready-made model selection
  - Simple House
  - Family House
  - Duplex Model
  - Luxury Villa
- House customization options
  - Budget
  - Material quality
  - Number of floors
  - Roof style
  - Bedrooms
  - Bathrooms
  - Doors
  - Windows
  - Wall color
  - Roof color
  - Door color
  - Parking section
  - Garden area
  - Furniture/appliances
- Requirement form for generating a model from user details
- Download options
  - JSON design file
  - Basic OBJ 3D model file
  - Project note text file
- Mobile responsive layout

## How to Run

Open `index.html` in a browser.

For the best result, use VS Code with the Live Server extension.

## Important Note

This is a front-end project demo. Login/signup data and house design data are saved in browser local storage. For a real project, connect it with PHP/Node.js backend and MySQL/MongoDB database.
